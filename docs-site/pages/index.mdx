# 바이브 코딩으로 수익나는 앱 만들기

세종청년센터에서 진행하는 **4일 집중 강의**의 학습 자료입니다. 수업 시간에 함께 보고, 수업이 끝난 뒤에도 다시 펼쳐 볼 수 있도록 정리했습니다.

오른쪽 상단의 검색창에서 키워드로 바로 찾을 수도 있고, 왼쪽 사이드바에서 Day 1 ~ Day 4 순서대로 따라가도 됩니다.

## 이 강의는 누구를 위한 강의인가요?

- 코드를 한 번도 써 본 적이 없는 비전공자
- ChatGPT를 한두 번 사용해 본 정도의 분
- "AI로 앱을 만든다"는 말은 들어 봤지만 어디서부터 시작해야 할지 모르는 분

코딩 경험이 없어도 괜찮습니다. 이 강의에서 우리가 배우는 것은 "코드를 직접 짜는 기술"이 아니라 **AI에게 잘 시키는 능력**입니다.

## "바이브 코딩"이 뭔가요?

**바이브 코딩(Vibe Coding)** 은 코드를 직접 한 줄 한 줄 타이핑하는 대신, AI에게 자연어로 "이런 거 만들어 줘"라고 시켜서 앱을 만드는 방식입니다.

비유하자면, 옛날에는 그림을 그리려면 붓을 잡고 직접 그려야 했지만, 이제는 "노을 지는 바닷가 풍경을 그려 줘"라고 말하면 AI가 그려 주는 시대입니다. 코딩도 똑같이 변하고 있습니다.

<Callout type="info">
바이브 코딩에서는 **"무엇을 만들고 싶은지 정확히 설명할 수 있는 능력"** 이 가장 중요합니다. 이건 비전공자가 오히려 잘하는 영역입니다.
</Callout>

## 솔직한 강의 목표

> "4일 만에 수익이 나는 상용 앱을 완성합니다"는 현실적인 목표가 아닙니다.
> 진짜로 가져갈 것은 다음 네 가지입니다.

1. **AI가 어디까지 왔는지** 손으로 직접 만져 보고 체감합니다.
2. 강의가 끝난 뒤에도 혼자 다시 시도할 수 있는 **워크플로우와 도구 목록**을 손에 쥡니다.
3. 본인 아이디어 기반의 **데모 앱 1개**를 직접 인터넷에 띄워 봅니다.
4. **수익화의 구조**를 이해합니다. (실제 입금까지는 사업자 등록 등 별도 절차가 필요합니다.)

## 사용하는 도구는 모두 무료입니다

이 강의에서 사용하는 도구는 모두 **무료 한도** 안에서 충분히 굴러갑니다. 신용카드 등록도 필요하지 않습니다.

| 분야 | 도구 |
| --- | --- |
| 메인 앱 빌더 | Lovable |
| 백업 빌더 | v0 (Vercel), Bolt (StackBlitz) |
| 챗봇 | ChatGPT, Claude, Gemini, Perplexity |
| 이미지 | Nano Banana(Gemini), Bing Image Creator, Ideogram |
| 음악·음성 | Suno, ElevenLabs |
| 영상 | Runway, Pika, Veo |
| 백엔드 | Supabase |
| 배포 | Vercel |
| 결제(시연용) | Stripe Payment Links, 토스페이먼츠 테스트 모드, Gumroad |

<Callout type="info">
무료 한도는 도구마다 다릅니다. 한도가 소진되더라도 당황하지 않도록, 각 챕터에 **백업 도구**를 함께 안내합니다.
</Callout>

## "끝까지 못 만들어도 괜찮습니다"

20명이 동시에 진행하다 보면 누군가는 진도가 늦어집니다. **완성보다 워크플로우**가 더 중요합니다. 오늘 못 만들었어도 이 문서를 보면서 내일 혼자 다시 만들 수 있다면, 이 강의의 목표는 달성된 것입니다.

옆사람과 화면을 공유하면서 진행해도 좋습니다. 막히면 짝꿍에게 먼저 물어보고, 그래도 안 풀리면 강사·보조 인력을 부르세요.
