# (번외) 어떤 도구를 결제할까 — 코드 수정형 AI 추천

## 이 페이지는 누구를 위한 것인가

- 강의 후에도 **계속 만들고 싶은** 분
- 4일 동안 Lovable 무료 한도가 모자라다고 느낀 분
- 본인 컴퓨터에 받은 코드를 **직접 수정**해 보고 싶은 분

## 핵심 메시지

> **이 강의는 무료 도구만으로 다 마칠 수 있게 설계됐습니다.**
> 그러나 강의 후에도 계속 만들 거라면, 가장 큰 도약은 **"코드를 직접 수정해 주는 AI"** 도구를 1개 결제하는 것입니다.

Lovable은 "한 메시지당 화면 단위로 시키는" 도구입니다. 그 다음 단계는 **본인 코드를 IDE/터미널에 띄워 두고 AI에게 자연어로 시켜 직접 편집**하는 도구입니다. 메시지 한도와 무관하게 거의 무한히 시킬 수 있고, 한 번에 여러 파일을 동시에 다룰 수 있습니다.

<Callout type="warning">
**가격·플랜 표기는 2026-05 기준입니다.** 자주 바뀌니, 결제 직전에 반드시 각 도구의 공식 가격 페이지를 다시 확인하세요. 무료 평가 기간이 있는 도구는 한 달만 써 보고 결정하는 것을 권장합니다.
</Callout>

## 1순위 — 코드 수정형 AI 중 1개 (월 $20대)

비전공자가 강의 후 **처음 결제하기 가장 강력한 1개**. Lovable이 만든 코드를 본인 컴퓨터로 받아서 AI에게 자연어로 직접 수정시킵니다.

| 도구 | 가격 (2026-05 기준) | 형태 | 비전공자 진입 장벽 |
| --- | --- | --- | --- |
| **[Cursor](https://cursor.com)** | Pro $20/월 | VS Code 기반 IDE | **가장 낮음** — 첫 결제 추천 |
| **[Claude Code](https://claude.com/claude-code)** | Claude Pro $20/월 (Max는 $100~$200/월) | 터미널 기반 + IDE 통합 | 중간 — 익숙해지면 가장 강력 |
| **[OpenAI Codex](https://platform.openai.com)** | ChatGPT Plus $20/월 또는 Codex CLI + API 종량제 | ChatGPT 안의 에이전트 또는 별도 CLI | 중간 |

### 어떤 걸 고를까 — 결정 흐름

```
본인 상황은?

├ "Lovable엔 익숙한데 IDE는 처음이야"
│   → Cursor
│   - VS Code 그대로라 화면 친숙
│   - 한국어 강의·블로그 자료 가장 풍부
│   - 자동완성 + 채팅 + Composer(여러 파일 동시 편집)
│
├ "터미널 명령어 한두 번 써 봤고 더 깊이 가고 싶어"
│   → Claude Code
│   - 코드베이스 전체 이해가 압도적
│   - 멀티 파일·멀티 단계 작업에 강함
│   - VS Code/Cursor/JetBrains 안에서도 호출 가능
│
└ "ChatGPT Plus를 이미 쓰고 있어"
    → OpenAI Codex
    - ChatGPT Plus($20/월)에 코드 에이전트가 포함
    - 추가 비용 없이 시작 가능
    - Codex CLI를 따로 깔아 본인 컴퓨터의 코드도 만질 수 있음
```

### 첫 1개로 가장 무난한 추천

세 도구 모두 강력하지만, **비전공자 첫 결제**라면 **Cursor** 가 가장 진입 부담이 작습니다. 화면이 VS Code 그대로라서 검색하면 한국어 튜토리얼이 가장 많이 나옵니다.

코드·터미널이 처음부터 부담 없으면 **Claude Code**가 다음 6개월의 천장이 가장 높습니다.

## 같이 고려할 만한 결제 — Lovable Pro

이미 강의에서 사용한 도구라 가장 자연스러운 업그레이드입니다.

| 항목 | Lovable Pro |
| --- | --- |
| 가격 | 월 $25 (연간 결제 시 월 $21) |
| 메시지 | 월 100 + 하루 5 (최대 150) |
| 커스텀 도메인 연결 | 가능 |
| Lovable 배지 제거 | 가능 |

다만 Lovable Pro는 본질적으로 **"메시지를 더 많이 사는 것"** 입니다. 능력의 확장이 아니라 사용량의 확장이에요. 그래서 1순위는 코드 수정형 AI 1개, 그 다음에 필요해지면 Lovable Pro를 얹는 순서가 효율적입니다.

## 추천 시나리오 — 본인 상황별

### 시나리오 A — "Lovable로 일단 끝까지 본다"

- **결제 1**: Lovable Pro ($25/월)
- **결제 2**: 없음 (3~6개월 후 다시 검토)
- **언제 좋은가**: 코드는 일단 안 보고 싶고, 화면 단위로 계속 시키는 흐름이 편한 분

### 시나리오 B — "코드 좀 만져 보고 싶다" ⭐ 강사 추천

- **결제 1**: Cursor Pro ($20/월) — 첫 한 달
- **결제 2**: 적응되면 Claude Code 추가 또는 전환
- **언제 좋은가**: "Lovable이 만든 코드가 어떻게 생겼는지 한번은 봐 보고 싶다" 싶을 때

### 시나리오 C — "본격 메이커로 간다"

- **결제 1**: Claude Code (Pro $20/월 → 부족하면 Max로)
- **결제 2**: Lovable Pro ($25/월) — 배포·도메인용
- **결제 3**: 도메인 (연 $10~20)
- **언제 좋은가**: 강의 후 1~2달 안에 본인 앱 정식 출시 계획이 있는 분

## 결제 후 첫 1주일에 할 일

처음 결제하면 어디서 시작해야 할지 막막합니다. 다음 4단계를 따라가면 손에 감각이 잡힙니다.

```
1. Lovable의 본인 프로젝트를 GitHub에 연결
   → Lovable의 "GitHub" 메뉴에서 한 번 클릭

2. 본인 컴퓨터에 코드 받기 (git clone)
   → 결제한 도구(Cursor 등)에서 그 폴더 열기

3. AI에게 첫 질문 던지기
   → "이 코드의 전체 구조를 한 화면 그림으로 설명해 줘"
   → "메인 화면이 어느 파일에 있어?"

4. 작은 수정 1개 시도 (예: 메인 화면 글자색 바꾸기)
   → AI가 어느 파일의 어디를 고쳤는지 보면서 따라가기
   → 저장 → git push → 배포에 자동 반영되는지 확인
```

이 한 사이클을 돌리면 **"AI에게 코드 시키기"가 어떤 감각인지** 머릿속에 들어옵니다.

## 결제하지 말아야 할 것 (지금 단계엔 불필요)

| 영역 | 왜 지금은 비추천 |
| --- | --- |
| 분석·모니터링 SaaS Pro 플랜 | 사용자 100명 넘기 전엔 무료 플랜으로 충분 |
| 영상 도구 Pro (Runway, Pika 등) | 결과 품질 대비 가격이 비쌈, 무료 체험으로 충분 |
| 디자인 도구 Pro (Figma, Framer) | 이미지 AI로 대체 가능, 정 필요해지면 그때 |
| 호스팅 Pro 플랜 | Lovable Cloud 무료 한도로 강의 후 수개월은 충분 |

처음엔 **코드 수정형 AI 1개 + (선택) Lovable Pro** 이 조합이면 충분합니다.

## 자주 묻는 질문

### Q. "Cursor와 Claude Code 둘 다 사면 너무 비싸지 않나요?"

처음엔 1개만 결제하세요. Cursor에 적응되면 그 IDE 안에서 Claude Code를 통합으로 얹는 게 자연스러운 흐름입니다. 둘 다 Pro 기준 합쳐서 월 $40 정도이니, 1~2달 써 본 뒤 본인이 진짜 자주 쓰는 쪽만 남기세요.

### Q. "강의 직후 바로 결제해야 하나요?"

**아니요.** 강의 후 1~2주는 무료로 더 시도해 보고, "Lovable 메시지 한도가 부족하다"는 답답함이 진짜로 들 때 결제하세요. 막연한 기대로 미리 결제하면 한 달 후에 거의 안 쓰는 경우가 가장 흔합니다.

### Q. "한국 카드로 결제 가능한가요?"

세 도구 모두 한국 신용/체크카드로 결제 가능합니다. 환율 + 부가세(VAT) 별도로 청구되어 실제 월 청구액은 표기 금액의 약 1,400원/달러 + 10% 부가세 정도로 계산됩니다.

### Q. "환불 가능한가요?"

- **Cursor**: 보통 결제 후 7일 이내 환불 가능 (공식 약관 재확인)
- **Claude (Anthropic)**: 케이스별 처리, 자동 환불 정책 제한적
- **ChatGPT Plus**: 일반적으로 환불 어려움

결제 전 카드로 **한 달치만** 결제하고 그 한 달 안에 결정하는 흐름이 안전합니다.

### Q. "강의에서는 왜 이 도구들을 안 썼나요?"

이 강의는 **결제 0원** 전제로 설계됐습니다. 비전공자 20명이 동시에 시작하기 가장 진입 장벽이 낮은 게 Lovable이라서 메인으로 골랐습니다. Cursor / Claude Code / Codex는 **강의 후에도 계속하실 분이 결제하면 도약이 큰** 도구들입니다.

### Q. "회사에서 비용 처리 가능한가요?"

세 도구 모두 영문 영수증 발급이 가능합니다. 부가세 처리를 위해 사업자 번호 등록이 가능한 도구도 있으니, 본인 회사 경리·세무 담당자에게 문의해 보세요.

## 마지막 한 마디

> **도구를 결제 안 한다고 못 하는 게 아닙니다.** 무료로도 충분히 갈 수 있습니다.
> 다만 "Lovable 메시지가 부족하다"는 답답함이 진짜로 들기 시작하면, 그때 코드 수정형 AI 1개를 결제하세요.
> 그 1개가 다음 6개월의 도약을 만듭니다.
