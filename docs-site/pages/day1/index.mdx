# Day 1 — AI 도구 만물상 투어

오늘은 **AI가 지금 어디까지 왔는지 직접 만져 보는 날**입니다. 텍스트, 이미지, 음악·음성, 영상, 그리고 마지막으로 노코드 앱 빌더까지 한 바퀴 돌아 봅니다.

## 오늘의 학습 목표

- AI가 지금 어디까지 왔는지 **직접 만져 보며 체감**합니다.
- 텍스트 / 이미지 / 음악·음성 / 영상 / 코드(노코드 앱 빌더)까지 한 바퀴 돕니다.
- "내가 다시 시도해 볼 수 있겠다" 싶은 도구를 **최소 2~3개** 챙깁니다.
- 바이브 코딩은 **코드를 짜는 게 아니라 AI에게 잘 시키는 능력**임을 이해합니다.

## 오늘 챙겨 갈 4가지 핵심 개념

### 1. 프롬프트 = 지시서

AI는 무한히 시킬 수 있는 인턴이지만, **명확하게 시켜야** 결과가 나옵니다. 같은 일이라도 어떻게 시키느냐에 따라 결과 품질이 천지 차이입니다.

### 2. 모든 AI는 다 다릅니다

같은 질문도 ChatGPT / Claude / Gemini의 답이 모두 다릅니다. 그래서 "어느 게 가장 좋아요?"보다 **용도별로 구분해서 쓰는 감각**이 중요합니다.

### 3. 무료 한도 안에서 살아남기

도구마다 무료 한도가 다르고, 강점·약점이 다릅니다. 한 도구가 막히면 다른 도구로 갈아탈 줄 알아야 합니다.

### 4. 결과물보다 워크플로우

오늘 만든 것보다 **"내일 혼자 다시 할 수 있는가"** 가 더 중요합니다. 그래서 이 사이트가 있는 것입니다.

## 오늘 자주 보일 함정

- **가입 단계에서 막힙니다.** 인증 메일이 안 오거나 CAPTCHA가 반복됩니다 → 잠깐 다른 도구 먼저 해 보고 5분 뒤 다시 시도하세요.
- **무료 한도가 빨리 떨어집니다.** Suno 10크레딧, Runway 영상 1~2개 등 → "체험은 1번만, 욕심내지 말기"가 원칙입니다.
