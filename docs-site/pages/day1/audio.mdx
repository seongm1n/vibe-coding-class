# 음악·음성 생성

## 이 챕터의 목표

AI로 노래(BGM)와 음성을 만들어 봅니다. 본인 앱 홍보용 BGM 한 줄과 안내 음성 한 문장을 만드는 게 목표입니다.

이 결과물은 Day 4의 **랜딩페이지**나 발표용 짧은 영상에 그대로 쓸 수 있습니다.

## 사용할 도구

| 도구 | 용도 | 무료 한도 |
| --- | --- | --- |
| [Suno](https://suno.com) | 가사 + 장르를 입력하면 노래 1곡 생성 | 일 10크레딧 (대략 노래 5곡 분량) |
| [ElevenLabs](https://elevenlabs.io) | 텍스트를 사람 목소리로 합성, 한국어 가능 | 월 10,000 크레딧 (한국어 기준 약 10분 분량) |

<Callout type="info">
**Suno는 진짜 가수가 부른 것처럼 노래를 만들어 줍니다.** 처음 들어 보면 다들 한 번씩 놀라는 결과물이 나옵니다.
</Callout>

## 다 같이 따라 하기

이 챕터는 **Suno로 곡 만들기 → ElevenLabs로 안내 음성** 흐름으로 갑니다.

### 사전 준비

브라우저에 **새 탭 2개**를 엽니다.

| 탭 | 사이트 | URL | 가입 |
| --- | --- | --- | --- |
| 1 | Suno | https://suno.com | Google 계정 권장 |
| 2 | ElevenLabs | https://elevenlabs.io | 이메일 인증 필요 |

ElevenLabs는 가입 시 **이메일 인증 메일**이 옵니다. 메일이 늦게 오면 일단 Suno부터 시작하세요.

### 1단계 — 가사 자동 생성

가사를 직접 짜는 것보다 **ChatGPT에 시키는 게** 훨씬 빠릅니다.

ChatGPT (또는 Claude) 탭에 다음을 붙여 넣으세요.

```text filename="앱 BGM 가사 생성 — 라이브"
나는 [본인 앱 이름] 이라는 앱의 짧은 홍보 BGM을 만들고 싶어.
[본인 앱이 뭘 해 주는지 한 줄] 한국어 가사로 30초 분량 써 줘.

요구사항:
- 후킹되는 후렴구가 1번은 들어가야 해
- 가사 안에 앱 이름이 1~2번 등장
- 가벼운 K-pop 또는 일렉 팝 분위기
- [Verse]와 [Chorus] 구조로 표시
- 너무 길지 않게 30초 정도 분량

추가로, Suno에 넣을 영어 Style 키워드도 1줄 같이 줘.
```

ChatGPT가 만들어 준 **(1) 가사** 와 **(2) 영어 Style** 두 가지를 메모장에 복사해 두세요.

#### 가사 예시 (운동 기록 앱 기준)

```text filename="ChatGPT 결과 예시"
[Verse]
오늘도 헬스장 가는 길
머뭇거리던 내가 달라졌어
FitCheck 한 번이면 끝
출석 체크 너무 쉬워

[Chorus]
FitCheck FitCheck
오늘도 너와 함께
FitCheck FitCheck
운동 습관 만들자

Style: upbeat K-pop, female vocal, electric guitar, energetic, 110 bpm, motivational
```

### 2단계 — Suno에 곡 만들기

#### 진입 흐름

1. **1번 탭(Suno)** 으로 이동
2. 좌측 사이드바에서 **"Create"** 클릭
3. 두 가지 모드 중 선택:
   - **Simple**: 한 줄 설명만 넣으면 자동 생성 (편하지만 디테일 약함)
   - **Custom**: 가사·Style·곡 제목을 직접 입력 (오늘 우리는 이걸 씁니다)
4. **"Custom"** 모드로 전환

#### 입력 필드

| 필드 | 무엇을 넣나 |
| --- | --- |
| **Lyrics** | 1단계에서 ChatGPT가 만든 가사 그대로 |
| **Style of Music** | 1단계의 영어 Style 키워드 |
| **Title** | 본인 앱 이름 (예: "FitCheck Anthem") |
| **Make Instrumental** | OFF (가사가 들어가야 함) |

#### 생성 클릭 → 1~2분 대기

Suno가 보통 **2곡을 동시에** 만들어 줍니다. 같은 가사·같은 Style이지만 멜로디는 다릅니다.

#### 결과 듣기

1. 좌측 사이드바 **"My Songs"** 또는 결과 화면에서 ▶️ 재생
2. **이어폰 / 헤드폰을 끼고** 들으세요. 강의실에 소리가 퍼지면 다른 분 작업에 방해가 됩니다.
3. 2곡 중 마음에 드는 1곡 선택

<Callout type="warning">
**WiFi 부하 주의**: 20명이 동시에 Suno를 돌리면 결과 생성이 평소보다 느릴 수 있습니다. 5분 넘게 기다려도 안 나오면 새로고침 한 번. 그래도 안 되면 **다음 단계로 일단 넘어가세요**, 백그라운드에서 계속 만들어집니다.
</Callout>

#### 다운로드

마음에 드는 곡 옆 **"..." 메뉴 → Download → MP3** 클릭. 본인 컴퓨터에 저장.

파일명: `app-bgm.mp3` 정도로 알아보기 쉽게.

### 3단계 — ElevenLabs로 안내 음성

이메일 인증이 끝났으면 ElevenLabs로 이동합니다. 안 됐다면 강사·보조 인력에게 알려 주세요.

#### 진입 흐름

1. **2번 탭(ElevenLabs)** → 로그인
2. 좌측 사이드바 **"Speech"** → **"Text to Speech"** 클릭
3. 가운데 큰 텍스트 영역이 보입니다.

#### Voice 선택

화면 우측 상단 또는 하단에 **"Voice"** 드롭다운이 있습니다. 클릭하면 다양한 목소리가 보입니다.

- 한국어가 자연스러운 Voice를 찾으세요. 보통 **"Multilingual"** 표시가 있는 것들이 한국어를 잘합니다.
- "Bella", "Rachel", "Antoni" 같은 이름이 보이면 한 번씩 미리듣기 (▶️) 해 보세요.
- 본인 앱 분위기에 맞는 목소리 1개 고르기

#### 스크립트 입력

다음 스크립트를 그대로 붙여 넣거나 본인 앱에 맞춰 수정하세요.

```text filename="ElevenLabs 안내 음성 스크립트"
안녕하세요, [본인 앱 이름]입니다.
오늘도 [짧은 인사 — 운동/식사/공부 등 앱 컨셉에 맞게].
아래 [버튼/입력창]을 눌러 시작해 주세요.
저희와 함께 [한 줄 약속].
```

#### 음성 생성

1. **"Generate"** 또는 ▶️ 버튼 클릭
2. 5~10초 대기 → 음성이 자동 재생됨
3. 발음이 어색한 부분이 있으면 **띄어쓰기**나 **한자어**를 살짝 수정 → 다시 Generate

#### 다운로드

음성 결과 옆 ⬇️ 다운로드 아이콘 클릭. MP3로 저장.

파일명: `app-voice.mp3`

### 4단계 — 본인 결과 다시 듣고 톤 정리

본인이 만든 곡과 음성을 한 번 더 들으면서 **본인 앱의 톤**을 정리해 두세요.

- 너무 차분한가, 너무 들떴는가?
- 다음에 만들 때 어떤 키워드를 추가/삭제하면 더 본인 앱과 맞을까?
- 가사·스크립트에서 살짝 바꾸고 싶은 한 줄이 있다면 메모해 두기

이 감각이 잡히면 Day 4 랜딩페이지에서 다시 듣고 결정할 때 빠르게 판단할 수 있습니다.

### 결과 정리

체크포인트:

- [ ] 본인 컴퓨터에 `app-bgm.mp3` 저장됨
- [ ] (선택) `app-voice.mp3` 저장됨
- [ ] Day 4 랜딩페이지·발표용으로 쓸 수 있게 한 폴더에 모아 둠

## 실습 (선택) — Suno로 더 다양한 시도

위 라이브가 끝나고 시간이 남거나 다음 날 혼자 다시 시도할 때 참고할 프롬프트 예시입니다.

### 프롬프트 예시 — 3가지 버전

#### 나쁜 예

```text filename="나쁜 예 (Style 입력)"
신나는 노래
```

```text filename="나쁜 예 (Lyrics)"
앱 좋다
```

장르도 모호하고 가사도 의미가 없습니다. 결과는 그냥 "신나는 노래" 정도가 무작위로 나옵니다.

#### 보통 예

```text filename="보통 예 (Style)"
밝고 경쾌한 K-pop
```

```text filename="보통 예 (Lyrics)"
FitCheck으로 매일매일 출석 체크
운동을 시작해 보자 함께
```

스타일과 가사가 들어가 있지만, 분위기·악기 구성·언어 같은 디테일이 없습니다.

#### 좋은 예

```text filename="좋은 예 (Style)"
upbeat K-pop, female vocal, electric guitar, energetic, 110 bpm, modern, motivational
```

```text filename="좋은 예 (Lyrics)"
[Verse]
오늘도 헬스장 가는 길
머뭇거리던 내가 달라졌어
FitCheck 한 번이면 끝
출석 체크 너무 쉬워

[Chorus]
FitCheck FitCheck
오늘도 너와 함께
FitCheck FitCheck
운동 습관 만들자
```

장르(K-pop), 보컬(여성), 악기(일렉기타), 분위기(에너제틱), 템포(110bpm)까지 명시했습니다. 가사도 [Verse], [Chorus] 구조가 잡혀 있어 노래 완성도가 높아집니다.

<Callout type="info">
**Style 입력은 영어가 잘 먹습니다.** 가사는 한국어로 써도 한국어로 부릅니다.
</Callout>

### 예상 결과 비교

| 프롬프트 | 결과 | 부족한 점 |
| --- | --- | --- |
| 나쁜 예 | 어디선가 들어 본 듯한 평범한 BGM | 본인 앱과 무관 |
| 보통 예 | 그럭저럭 들을 만한 노래 1곡 | 분위기가 의도와 다를 수 있음 |
| 좋은 예 | 광고용으로도 쓸 수 있을 만한 완성도 | (없음 — 다음 단계는 가사·길이 미세 조정) |

### 개선 방법

- 장르를 너무 모호하게 적지 않습니다. 구체적인 장르명이 좋습니다 (예: "lo-fi hip hop", "synth pop", "acoustic ballad")
- 분위기를 형용사로 추가합니다 (energetic, calm, dreamy, dark, hopeful 등)
- 길이는 보통 1~2분 정도가 자동으로 잡히니, 너무 길게 만들 필요가 없습니다
- 결과가 마음에 안 들면 **같은 가사로 Style만 바꿔서** 다시 생성하면 크레딧을 아낄 수 있습니다

### 체크포인트

- [ ] Suno에서 본인 앱 이름이 들어간 노래 1곡 생성됨
- [ ] 다운로드 버튼으로 mp3 파일을 저장 (Day 4 랜딩페이지에서 사용 가능)

## 실습 2 — ElevenLabs로 안내 음성 만들기

### 실습 목표

앱의 첫 화면에 흐를 만한 **안내 음성 한 문장**을 만듭니다.

### 준비물

- ElevenLabs 가입 (이메일 인증 필요)

### 프롬프트(스크립트) 예시

#### 나쁜 예

```text filename="나쁜 예"
앱 안녕
```

너무 짧고 어색합니다. 자연스러운 톤이 안 나옵니다.

#### 보통 예

```text filename="보통 예"
안녕하세요, FitCheck입니다. 출석을 체크해주세요.
```

문장은 자연스럽지만 톤이 평범합니다.

#### 좋은 예

```text filename="좋은 예"
안녕하세요, FitCheck입니다. 오늘도 운동하러 와주셔서 감사합니다. 아래 버튼을 눌러 출석을 체크해주세요. 같이 건강해져요.
```

자연스럽고 따뜻한 톤이 살아납니다. 길이가 적당해서 음성으로 들었을 때도 어색하지 않습니다.

### 음성(Voice) 선택 팁

ElevenLabs는 다양한 목소리를 고를 수 있습니다.

- **남성/여성** 선택
- **나이대** (어린/젊은/중년)
- **톤** (따뜻한/차분한/에너지 넘치는)

본인 앱의 분위기에 맞춰 골라 보세요. 같은 스크립트도 목소리에 따라 완전히 다른 느낌이 납니다.

### 개선 방법

- 음성이 너무 빠르거나 느리면 **속도 조절** 슬라이더로 조정합니다
- 어색한 발음이 있으면 **띄어쓰기**를 바꿔 보거나 한자어를 한국어로 바꿉니다
- 목소리가 마음에 안 들면 다른 Voice로 다시 생성합니다

### 체크포인트

- [ ] 30초 미만의 mp3 파일 1개 저장됨
- [ ] 한국어 발음이 자연스러움

## 자주 묻는 질문

### Q. "AI가 만든 노래도 저작권이 있나요?"

도구마다 다릅니다. **Suno 무료 플랜은 비상업적 사용만 허용**됩니다. 상업적으로 쓰려면 Pro 플랜이 필요합니다. 강의 데모용으로는 무료로 충분합니다.

### Q. "내 목소리로 안내 음성 만들 수 있어요?"

ElevenLabs 유료 플랜에서 **음성 클로닝(Voice Cloning)** 기능이 있습니다. 본인 목소리 30초 정도를 녹음해 올리면 그 목소리로 어떤 문장이든 합성해 줍니다. 강의에서는 무료 음성으로 충분합니다.

### Q. "노래 가사를 자동으로 만들고 싶어요"

Suno 안에 가사 생성 기능이 있습니다. 또는 ChatGPT에 "FitCheck 앱을 위한 30초 분량의 K-pop 가사를 [Verse], [Chorus] 구조로 써 줘"라고 시킨 뒤 Suno에 붙여 넣으세요.
